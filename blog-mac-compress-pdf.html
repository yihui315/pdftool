PDF压缩教程

Mac 怎么压缩 PDF 文件？macOS 自带工具 + 在线压缩方法

更新于 2026 年 7 月 · 阅读时间约 8 分钟

Mac 电脑用户处理 PDF 文档时，往往不需要额外安装任何软件——macOS 系统已经内置了强大的 PDF 处理能力，其中就包括 PDF 压缩功能。无论你是用苹果自带的预览.app，还是通过浏览器使用 pdftool.work 的在线工具，都可以在 Mac 上快速完成 PDF 压缩。本文将详细介绍几种实用的 Mac PDF 压缩方法，帮助你找到最适合自己工作流程的方案。

方法一：使用 macOS 预览.app 压缩 PDF（推荐）

预览.app 是 macOS 系统自带的查看和编辑工具，不仅可以查看图片和 PDF，还内置了 PDF 压缩功能。对于大多数日常办公文档，这个方法完全免费且足够使用。

操作步骤

  1. 在 Finder 中找到需要压缩的 PDF 文件，双击用预览.app 打开（也可以右键点击文件 → 「打开方式」→「预览」）。
  2. 在预览.app 顶部菜单栏点击「文件」→「导出 PDF...」。
  3. 在导出面板中，可以看到「Quartz 滤波器」下拉菜单。点击展开菜单，选择「Reduce File Size」选项（这是 macOS 内置的 PDF 压缩滤波器）。
  4. 选择保存位置，点击「存储」即可生成压缩后的 PDF 文件。

关于 Quartz 滤波器

macOS 提供了多个 Quartz 滤波器，其中最实用的两个是：

  • Reduce File Size：通过降低图片分辨率和移除元数据来减小文件体积，压缩效果显著，适合文件体积较大但对画质要求不极端的场景。
  • Optimize Color PDF / Optimize Grayscale PDF：更温和的压缩方式，适合需要保持相对较高画质但仍需减小体积的场景。

预览.app 压缩的局限性

预览.app 的压缩功能虽然方便，但选项相对单一，无法精确控制压缩级别。如果需要更精细的控制（如指定压缩到特定大小、保留特定图片质量），建议使用 pdftool.work 在线压缩工具，它提供轻度、中度、极限等多个压缩级别可选。

方法二：使用 Adobe Acrobat Reader Mac 版

Adobe Acrobat Reader 是最流行的 PDF 阅读器，Mac 用户可以免费从 Mac App Store 下载安装基础版本。

基础版 vs 付费版

  • Acrobat Reader（免费版）：支持查看、注释、签名等基础功能，但不提供 PDF 压缩功能。
  • Acrobat Pro（付费版）：提供完整的 PDF 编辑、压缩、优化和转换功能。如果你经常需要专业处理 PDF，Acrobat Pro 是最强大的选择，但价格较高（按年订阅）。

如果你不想付费，pdftool.work 的免费在线工具提供了与 Acrobat Pro 相当的压缩效果，是一个很好的替代方案。

方法三：使用 pdftool.work 在线压缩工具

这是最推荐的Mac PDF 压缩方法之一。通过 Safari 或 Chrome 等浏览器访问 pdftool.work，整个压缩过程在浏览器本地（JavaScript）完成，文件不会上传到服务器。

操作步骤

  1. 在 Mac Safari 或 Chrome 浏览器中打开 pdftool.work PDF 压缩工具。
  2. 点击上传按钮，在文件选择器中选择需要压缩的 PDF 文件。
  3. 上传后，工具会显示原文件大小，并提供多个压缩级别：轻度压缩（推荐）、中度压缩、极限压缩。
  4. 选择合适的压缩级别，点击「压缩」按钮。浏览器会在本地处理文件。
  5. 压缩完成后，页面显示压缩后的文件大小和压缩率。确认效果后点击「下载」保存文件。

与预览.app 的对比

  • 优势：提供多个压缩级别，可根据需求灵活选择；界面更直观，压缩效果实时可见；支持批量压缩；支持与 PDF 转图片、PDF 分割等工具联动使用。
  • 共同点：完全免费；文件在本地处理，不上传服务器。

方法四：使用 Mac Automator 自动化压缩

如果你经常需要批量压缩 PDF，可以通过 Mac 自带的 Automator（自动操作）应用创建一个自动化工作流程，一键压缩文件夹中的所有 PDF。

创建 Automator 工作流程

  1. 打开「应用程序」→「Automator」。
  2. 选择「服务」（Service）类型的工作流程。
  3. 在左侧操作库中找到「PDF」分类，将「压缩 PDF 文档」操作拖拽到右侧工作区。
  4. 保存工作流程，并为其命名（如「压缩 PDF」）。
  5. 之后在 Finder 中右键点击 PDF 文件，选择「快速操作」或「服务」→「压缩 PDF」即可一键压缩。

Automator 压缩效果

Automator 的「压缩 PDF 文档」操作本质上调用的是 macOS 的系统滤波器，效果与预览.app 的「Reduce File Size」相近。对于需要批量处理的用户，这是一个节省时间的好方法。

Mac 压缩 PDF 常见问题

Mac 压缩 PDF 后文件变模糊怎么办？

这是「Reduce File Size」滤波器的已知问题——它会大幅降低图片分辨率以减小体积，导致图片和扫描件变得模糊。解决方案是使用更温和的压缩方式，如 pdftool.work 的「轻度压缩」选项，或者在预览.app 中选择「Optimize Color PDF」滤波器而非「Reduce File Size」。

Mac 有没有无损压缩 PDF 的方法？

macOS 的 Quartz 滤波器中没有严格意义上的「无损」压缩，因为所有压缩都会对 PDF 内容进行重新编码。但「Optimize Color PDF」相比「Reduce File Size」对画质的损耗要小得多。如果你追求最高画质，建议在压缩前保留原始文件副本，并选择轻度压缩级别。

压缩后的 PDF 在其他设备上能正常打开吗？

可以。PDF 是通用格式，压缩后的文件在任何设备（Windows PC、安卓手机、iPhone 等）上都能正常打开。如果在某些老旧的 PDF 阅读器中打开出现问题，建议使用通用性更好的 pdftool.work 压缩工具，它的兼容性优化更好。

MacBook Air / Mac mini 等低配 Mac 适合用浏览器压缩大 PDF 吗？

适合。pdftool.work 的压缩处理完全在浏览器本地进行，不依赖服务器算力，即使是低配 Mac 也能流畅处理。但需要注意，压缩较大的扫描版 PDF（几十 MB 以上）可能会占用较多内存，建议分批处理。

Mac PDF 压缩最佳实践

  • 保留原始文件：压缩前先复制一份原始 PDF 作为备份。压缩后的文件无法还原到原始质量。
  • 选对压缩方式：普通办公文档（文字为主）用预览.app 的 Reduce File Size 即可；包含图片的文档建议用 pdftool.work 的轻度压缩以保护画质。
  • 批量处理用 Automator：如果经常需要压缩大量 PDF，建立 Automator 工作流程可以显著提升效率。
  • 压缩前优化源文件：如果 PDF 是自己用 Word 或 Keynote 导出的，可以先将图片素材压缩到合适分辨率再导出，这样最终 PDF 体积更小，也避免了二次压缩对画质的影响。
  • 发送前再检查：压缩后的 PDF 发给对方前，建议自己先打开确认内容完整、清晰可读，尤其是涉及合同、证书等重要文件时。
