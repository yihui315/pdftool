Privasi

Privasi

Kebijakan ini menjelaskan pemrosesan PDF berbasis browser di PDFTool.work serta permintaan analitik, iklan, dan jaringan biasa yang dapat terjadi saat situs berjalan.

Terakhir diperbarui: 5 Juli 2026

1. Cara kerja pemrosesan file

Saat Anda memilih PDF di alat, isi file dibaca oleh browser agar halaman dapat membuat hasil di perangkat Anda. PDFTool.work dirancang agar tidak menerima isi PDF yang dipilih untuk operasi alat ini.

Hasil biasanya dibuat sebagai tautan unduhan sementara di browser. Jika Anda menyegarkan halaman, menutup tab, atau memilih file lain, tautan hasil mungkin tidak lagi tersedia.

2. Informasi yang mungkin diproses situs

Untuk menjaga situs tetap berjalan, memecahkan masalah, dan mendukung layanan gratis, server atau layanan pihak ketiga dapat memproses informasi non-isi-file seperti:

  • Log akses dasar, seperti alamat IP, jalur permintaan, jenis browser, waktu kunjungan, dan kode status.
  • Informasi kinerja dan kesalahan halaman, seperti kegagalan pemuatan skrip, kemampuan browser, atau tahap pemrosesan.
  • Cookie, informasi perangkat, atau data permintaan web biasa yang diperlukan oleh layanan analitik dan iklan, bergantung pada konfigurasi dan aturan yang berlaku.

3. Google Analytics dan Google AdSense

Situs dapat memuat sumber daya Google Analytics dan Google AdSense untuk memahami kunjungan dan mendukung operasi gratis. Layanan tersebut dapat menerima informasi permintaan web biasa, tetapi tidak perlu membaca isi file lokal yang dipilih di alat PDF.

4. Batas memori browser dan struktur PDF

Pindaian besar, PDF dengan banyak gambar, file terenkripsi, atau struktur PDF kompleks dapat memakai lebih banyak memori browser dan mungkin gagal diproses. Gunakan browser desktop untuk file penting dan tinjau hasil sebelum mengirimkannya.

5. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan ini, konfigurasi iklan, atau pemrosesan file, kirim email ke privacy@pdftool.work.
