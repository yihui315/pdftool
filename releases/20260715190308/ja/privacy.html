プライバシー

このポリシーは、PDFTool.work のブラウザベース PDF 処理と、サイト実行中に発生する可能性がある通常の解析、広告、ネットワークリクエストについて説明します。

最終更新日: 2026年7月5日

1. ファイル処理の仕組み

ツールで PDF を選択すると、ページが端末上で結果を作成できるよう、ファイル内容はブラウザによって読み取られます。PDFTool.work は、これらのツール操作で選択した PDF 内容を本サイトのサーバーが受け取らない設計です。

結果は通常、一時的なブラウザのダウンロードリンクとして作成されます。ページ更新、タブを閉じる、別のファイルを選ぶと、結果リンクが利用できなくなる場合があります。

2. サイトが処理する可能性のある情報

サイト運営、問題調査、無料サービス維持のため、サーバーまたは第三者サービスは、ファイル内容ではない次のような情報を処理する場合があります。

  • IP アドレス、リクエストパス、ブラウザ種類、訪問時刻、ステータスコードなどの基本アクセスログ。
  • スクリプト読み込み失敗、ブラウザ能力、処理段階など、ページの性能やエラー情報。
  • 設定や適用ルールに応じて、解析および広告サービスに必要な Cookie、端末情報、通常の Web リクエストデータ。

3. Google Analytics と Google AdSense

サイトは訪問状況の理解と無料運営の支援のため、Google Analytics や Google AdSense リソースを読み込む場合があります。これらのサービスは通常の Web リクエスト情報を受け取ることがありますが、PDF ツールで選択したローカルファイル内容を読む必要はありません。

4. ブラウザメモリと PDF 構造の制限

大きなスキャン、画像の多い PDF、暗号化ファイル、複雑な PDF 構造は、より多くのブラウザメモリを使い、処理に失敗する場合があります。重要なファイルはデスクトップブラウザで処理し、提出前に結果を確認してください。

5. 連絡先

このポリシー、広告設定、ファイル処理について質問がある場合は privacy@pdftool.work までメールしてください。
