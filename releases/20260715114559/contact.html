常见问题

PDF 合并/分割失败怎么办？

请确保 PDF 文件未损坏且不是扫描件图片合成的无文本 PDF。如问题持续，可尝试换用 Chrome 或 Firefox 最新版浏览器。

文件会上传到服务器吗？

不会。所有工具在浏览器本地处理，文件不会离开你的设备。详见 隐私政策。

压缩后的 PDF 模糊怎么办？

压缩算法会优先保留可读性。若图片质量下降明显，可尝试调整压缩目标大小，或使用 PDF 转图片 单独处理。

可以批量处理多个文件吗？

PDF 合并支持一次性选择多个文件合并为一个。分割支持按页码范围拆分。如需更复杂的批量处理，可以分步操作或 联系人工服务。

有 API 或开发者合作吗？

目前主要面向个人用户。如有企业级合作或定制开发需求，欢迎在邮件中说明，我们会安排对应同事跟进。
