Limits

A rejected upload can have more than one cause

Size is common, but browser memory, encrypted PDFs, page orientation, or platform rules can also block submission.

  • A file may be under the size limit but still fail because it is encrypted or malformed.
  • Large scans can exceed browser memory during processing.
  • Some platforms require a specific file type, page count, or readable resolution.

FAQ

PDF upload size questions

Why does my PDF upload fail even after compression?

The platform may also check page count, encryption, file structure, or readability, not only size.

Should I remove pages before compressing?

If pages are unnecessary, removing them first can be more reliable than compression alone.

Can I prepare files on a phone?

Small files may work, but a desktop browser is safer for important PDFs because mobile memory is limited.
